## Setup

```sh
git clone git@github.com:MasashiFukuzawa/vite-ffmpeg-wasm-sample.git
cd vite-ffmpeg-wasm-sample
yarn
yarn dev
# from another terminal
open http://localhost:5173/
```

Enjoy!
